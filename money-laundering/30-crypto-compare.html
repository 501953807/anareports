匿名性从高到低排列。执法追踪难度越高意味着洗钱者越偏好该币种。USDT因Tether可远程冻结合约地址资金，反而不适合大额长期持有，但因其流动性好仍被广泛用于短期洗钱通道。

数据信号：2024年加密货币相关洗钱量达$780亿，占全球洗钱总量的~15%。其中BTC占比最高(35%)因其流动性，但XMR虽然仅占8%的洗钱份额，执法追踪难度却是最低的。关键趋势：稳定币(USDT+USDC)洗钱占比从2020年的12%上升至2024年的25%，反映"快速出清"需求的增长。

五学派交叉分析：加密币种洗钱选择的经济学逻辑

Monetarism(货币主义)：洗钱者选择BTC还是XMR本质上是"货币流通速度vs货币匿名性"的权衡。Friedman的MV=PQ框架在此变形为：高流动性币种(BTC/ETH)追求V最大化(快速流转)，高匿名币种(XMR)追求Q的不可观测性。USDT作为稳定币，其V值极高但P(价格)固定，使其成为短期洗钱的理想载体——价值不会在流转中波动。
Supply-side Economics(供给侧经济学)：不同币种的"反洗钱供给弹性"差异巨大。BTC链上全透明，AML供给充足；XMR天然匿名，AML供给几乎为零。供给侧视角的洞见是：监管政策提高某类币种的AML成本(如CEX下架XMR)会刺激该币种转入更隐蔽的供给渠道(OTC/P2P)，反而可能增加整体风险。
Behavioral Economics(行为经济学)：洗钱者的币种选择受"框架效应"(framing effect)影响极大——将非法资金包装为"赌博收益"(赌场)、"投资增值"(钻石)、"技术中立"(BTC)都利用了认知偏差。USDT的高流动性让洗钱者产生"安全幻觉"，而XMR的极低流动性又让部分洗钱者因"损失厌恶"而回避，尽管其匿名性最优。
Ecological Economics(生态经济学)：从资源流动角度看，每种加密币种的洗钱路径都是一个独立的"信息生态系统"。BTC是开放透明的"浅水区"——所有交易可见但可被过滤；XMR是封闭的"深水区"——完全不可见。洗钱者在两个生态系统间的迁移成本(转换费用、时间成本、技术门槛)决定了整个系统的稳定性。
World Economic Cycle Theory(世界经济周期理论)：加密洗钱币种偏好与经济周期高度相关。2017年牛市：BTC主导(35%+)；2021年DeFi牛市：ETH和稳定币崛起；2024-2025监管收紧期：XMR份额下降但OTC市场扩大。康波周期视角：当前处于第六轮技术革命(AI+区块链)的上升期，洗钱手法将从单一币种向跨链混合演进。
Tornado Cash创始人Pertsev案 — 隐私工具的法律边界

2023年12月，荷兰裔俄罗斯开发者Roman Pertsev被判86个月监禁。

核心争议：Tornado Cash是"技术中立"的工具还是"洗钱共谋"？美国OFAC首次将去中心化智能合约列为制裁对象。

后果：ETH混币量骤降70%，但XMR和OTC市场交易量上升150%，证明"堵疏失衡"会导致洗钱转移而非消失。

法学界分裂：技术自由派认为代码即言论，合规派认为明知被用于洗钱仍提供服务构成共谋。

分化链：加密洗钱币种选择的博弈演化
起点 → BTC为主力洗钱币种 → OFAC制裁Tornado Cash(2022) → 分化A：ETH混币量↓70% → 洗钱者转向XMR → 终态A：XMR被CEX大规模下架 → OTC市场暴涨
分化B：USDT流动性优势 → 稳定币发行商(Tether)被要求配合冻结 → 终态B：USDT洗钱成本上升 → 多稳定币分散策略兴起
分化C：隐私币XMR天然匿名 → 但流动性极差 → 终态C：ZK-proof隐私币(如Aztec)出现 → 合规+隐私兼得？

合规操作指南：加密资产币种级风控策略

VASP/CEX风控团队的币种级准入与监控标准

不同加密币种的风险特征差异极大。合规团队应为每种交易币种设定明确的风控参数和监控规则。

  1. 币种风险分级：按匿名性/流动性/可追踪性三维评分将币种分为红(禁止交易)/黄(强化监控)/绿(标准监控)三色。BTC/ETH → 黄(需链上标签+风险评分)；XMR/ZEC/GRIN → 红(禁止充提/交易)；USDC → 绿(强合规发行人可快速审批)；USDT → 黄(可冻结但流动性最好，需重点监控)。
  2. 充值地址预筛查：客户申请充提某一币种时，自动查询该地址的历史交易路径(最多10跳)。若发现与混币器/暗网/制裁地址关联 → 禁止充值或要求二级KYC(视频面谈+地址用途声明)。
  3. 交易监控规则差异化：对不同币种配置不同的监控阈值。(a) BTC/ETH：单笔≥$50k自动风险标签查询；(b) USDT(TRC20)：单笔≥$30k触发；(c) USDC：单笔≥$100k触发(因其强合规属性)；(d) SOL/DOT：单笔≥$50k触发。
  4. 提现审批流程：大额提现(≥客户风险等级限额)必须执行(1)链上目的地地址风险标签；(2)是否为CEX/托管钱包/非托管钱包判断；(3)若目的地为CEX → 检查该CEX是否在FATF Travel Rule合规名单上。
