第六章：易损件与保养周期 — 全生命周期维护指南

新能源汽车和电动两轮车的保养体系差异巨大。前者以电控系统为核心，后者以机械易损件为主。以下按保养周期分层梳理。

6.4 DIY保养实操教程

掌握基础DIY保养技能可以节省30%-50%的维修费用。以下内容按操作难度从低到高排列，新手建议先从轮胎和刹车开始。

操作前请阅读：所有DIY操作均涉及安全相关部件。如果你对某一步骤没有把握，请停止操作并前往专业维修点。涉及电机/控制器/电池内部的拆解，强烈建议交由专业人士处理。

任务一：更换轮胎(真空胎)

工时费参考：路边店换轮胎人工费30-50元/条，自己换省下这笔钱。真空胎比有内胎的轮胎更麻烦一点，但一旦学会其实很简单。

所需工具

  • 撬胎棒(3根，约15元一套，不建议用螺丝刀替代——会刮坏轮毂)
  • 气动打气筒或电动充气泵(压力≥100PSI)
  • 肥皂水(轮胎与轮毂的密封处涂肥皂水辅助安装)
  • 千斤顶或大木块(支撑车身)

操作步骤

  1. 拆下车轮：松开轴螺母(通常用17mm或19mm套筒)，卸下左右侧压板。注意拍下电线连接位置(速度传感器/ABS线)，然后拔下插头
  2. 放尽气压：按住气门芯把气放完。真空胎的胎唇通常咬得很紧
  3. 撬开胎唇：用撬胎棒勾住胎唇边缘，沿着轮毂槽慢慢撬出。一侧先完全撬出来，然后撬另一侧。遇到顽固部位可以用肥皂水润滑
  4. 取下旧胎：将轮胎从轮毂上完全剥离。检查轮毂内壁是否有锈蚀或毛刺(用细砂纸打磨)
  5. 安装新胎：先在轮胎内壁和轮毂接触面涂少量肥皂水。将胎唇对准轮毂槽口，用手慢慢推进。最后一段最紧，用撬胎棒小心地把胎唇翻过轮毂边缘
  6. 充气定型：充气到初始状态时你会听到"砰砰"两声——这是胎唇咬合进轮毂密封槽的声音。如果没有听到，说明没有装到位，放气重新调整
  7. 调至标准气压：参考胎侧标注(通常前轮200kPa/后轮250kPa)。充气后转动轮子检查是否居中，如有偏移重新放气调整

任务二：更换刹车片/刹车鼓

电动车刹车片是耗材中的耗材，一个勤骑的人半年就得换一次。成本20-40元，维修店收费50-80元(含工时)。

盘式刹车(碟刹)更换步骤

  1. 判断更换时机：刹车片厚度≤2mm时必须更换。目测刹车卡钳侧面的观察窗，如果看不到摩擦材料了就该换了
  2. 拆卸卡钳：用内六角扳手卸下卡钳固定螺栓(通常是5mm内六角)，将卡钳从刹车盘上取下，挂在车架上(不要捏刹车手柄，否则卡钳活塞会弹出)
  3. 更换刹车片：取出旧刹车片，安装新片。注意摩擦面朝外，背面的金属弹片和定位销也要一起换上
  4. 归位卡钳活塞：如果旧片已经磨得很薄，新片装上去后活塞行程不够。用C型夹或专用工具将活塞缓缓压回卡钳体(注意不要用力过猛损伤密封圈)
  5. 重新安装：将卡钳装回车叉支架，拧紧螺栓(扭矩建议25-30Nm)。捏几次刹车手柄让活塞贴合新刹车片
  6. 磨合：新刹车片需要30-50km的磨合期，初期制动力只有80%，这是正常的

鼓式刹车更换步骤

  1. 拆下后轮：鼓刹在后轮上，需要先卸车轮才能看到刹车鼓总成
  2. 打开后盖：鼓体侧面有一个检修小盖板(通常用一个螺丝固定)，拧开后可以看到刹车蹄
  3. 更换刹车蹄：旧蹄摩擦面厚度<1mm就该换。直接抽出旧蹄、换上新蹄(注意弹簧挂钩位置)
  4. 调整间隙：后盖上有调整螺钉，顺时针转减小间隙(刹车更灵敏)，逆时针增大。调到捏把手时有2-3mm的空行程最佳

任务三：更换铅酸电池组

电池更换是最简单的DIY操作之一。整件事的核心要点：拆线前拍照、装线时对照、螺丝一定拧紧。

  1. 购买同规格电池：看旧电池标签上的电压(V)和容量(Ah)。如"6-DZF-20"=12V20Ah铅酸电池。通常4块串联成48V、5块成60V、6块成72V
  2. 断开旧电池：先拆负极(-)，再拆正极(+)。拆之前用手机拍一张接线照片(哪根线接哪个极、串联顺序)
  3. 取出旧电池：打开电池仓，把旧电池一块块取出。铅酸电池很沉(20Ah一块约18kg)，注意腰部发力，必要时两人配合
  4. 清洁电池仓：用干布擦去灰尘，用湿布(拧干)擦拭端子接触面去除氧化层
  5. 安装新电池：按照片上的接线顺序串联。先串联(电池1正→电池2负→电池3正…)最后留出总正负极。放入电池仓并用扎带固定
  6. 连接整车线束：总正极接红色线插头，总负极接黑色线插头。端子必须压紧——松动的端子会导致打火和发热
  7. 首次充电：新电池建议先充满再用(通常6-8小时)。第一次充饱之后激活电芯化学反应，后续才能达到标称容量

锂电池更换注意事项

锂电池更换相对简单，但有几个关键点不同：

  • 锂电池通常是一体化封装的(不能单个替换)，拆下整体换整体
  • 必须有BMS保护板，买替换电池时确认BMS参数与原车匹配(过充/过放/短路/过流/温度保护)
  • 锂电池接线可能有航空插头(防水接头)，拆卸时捏住插头本体拔出，不要拽电线
  • 更换后检查仪表显示的电压是否与标称一致(60V电池组满电约73V，欠压保护约52V)

任务四：电机检修与更换

电机是整个车上最不容易坏的部件之一，大多数情况不需要拆。但如果出现异响、无力、发热异常等问题，需要排查。

故障诊断三步法

  1. 听声音：骑行时电机内发出"哒哒哒"的金属撞击声 → 轴承损坏(需更换轴承，约30-50元)
    安静但有规律的"嗡嗡"声 → 正常(霍尔传感器PWM声)
  2. 摸温度：骑行30分钟后电机外壳温度：≤60℃正常，60-80℃偏高，>80℃异常(可能是磁钢退磁或绕组短路)
  3. 摇一摇：支起大撑，用手晃动后轮。轴向间隙>1mm → 轴承磨损，需要调整预紧力或更换轴承

电机更换实操

  1. 确认电机规格：看电机侧面的铭牌——电压(48V/60V/72V)、功率(400W/800W/1200W)、转速(极数)、轴径(通常19mm或22mm)、齿数(如果是齿轮电机)
  2. 拔下电机三相线：粗的三根线(红黄蓝)是电机相线，细的五根线(红黑白黄绿)是霍尔线。拔掉控制器端的插头
  3. 拆卸轮毂电机：卸下车轮轴螺母 → 取出车轮 → 注意不要扯断ABS线(如果有) → 电机从车架取出
  4. 安装新电机：反顺序装回，先装轮后接线。三相线顺序不重要(接反了调换方向)，但霍尔线必须对应插好
  5. 调试：装车后先不捏刹车，空转电机看是否顺畅。然后用控制器自检功能(雅迪/爱玛的操作方法是拧满油门3秒+满刹车3秒交替，具体看手册)

任务五：日常保养——每月5分钟

以下5项每月花5分钟检查一下，能避免80%的突发故障：

1. 胎压检查

  • 用胎压计测量，不足时补足
  • 前轮2.0Bar / 后轮2.5Bar(载重大则后轮加0.2-0.3)
  • 气压偏低→续航里程减少5-10%
  • 气压过高→抓地力下降+爆胎风险

2. 刹车检查

  • 捏刹车手感：自由行程≤15mm为佳
  • 目测刹车片厚度(盘刹)≥2mm
  • 鼓刹异响或制动力衰减 → 调间隙或换蹄
  • 刹车油(液压碟刹)：液面低于MIN线即添加DOT3/DOT4

3. 充电习惯

  • 铅酸：用完即充，每次充满。勿过充(超过12小时)
  • 三元锂：随用随充，保持在20%-80%区间最佳
  • 磷酸铁锂：每周至少一次充满(校准BMS)
  • 充电器接口干燥无水渍，雨后先擦干再充电

4. 电气检查

  • 所有灯光(大灯/尾灯/转向灯/刹车灯)是否正常工作
  • 喇叭声音够不够大(法规要求90-110dB)
  • 仪表显示是否正常(无乱码/缺划)
  • 各防水接头密封完好(尤其是控制器下方的)

5. 紧固件

  • 后视镜螺丝是否松动
  • 龙头转向轴预紧力(骑行时不飘)
  • 脚踏/货架/尾箱螺丝
  • 车轮轴螺母(尤其是经常走烂路的)

6. 冬季特别保养

  • 续航衰减是正常的(铅酸-30%，三元锂-40%)
  • 电池存放在温暖室内可恢复部分容量
  • 刹车片冬季霜天初期打滑，使用前先低速试刹
  • 防水接头结冰时不要强行拔插，化冰后再操作
