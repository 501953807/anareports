NEV产业集群深度 — 四大集群的竞争格局与差异化

中国NEV产业形成"长三角(最完整)+珠三角(最智能)+中西部(成本最低)+京津冀(政策中心)"四大集群。每个集群依托原有产业基础和政策优势，形成了差异化的竞争力。

城市核心企业产业定位产值(亿)
上海特斯拉/蔚来/上汽/拼多多(物流)整车+智驾+互联网~3,000
常州宁德时代基地/理想汽车电池+整车~1,500
合肥比亚迪/蔚来(中国总部)/江淮/联电整车+芯片+显示~2,000

宁波/台州吉利(杭州湾)/拓普集团/旭升民营车企+压铸+底盘~2,500
杭州吉利/零跑/网易(智驾)整车+AI~1,200
苏州/无锡上汽(苏州)/磊知智能零部件+智能化~1,000

长三角是中国NEV产业最密集的集群。仅常州一地就聚集了宁德时代基地+理想汽车工厂+上游数百家供应商，形成了"半小时供应链圈"。合肥靠"蔚来救命钱"一战成名，现已吸引比亚迪/联电/巨腾入驻。

广州小鹏/广汽埃安/希音(AI供应链)智能电动车+AI
东莞德赛西威/电池配套零部件+智能制造

珠三角的核心优势是"电子+互联网+智能"基因。华为(深圳)提供全栈智驾解决方案，大疆车载(桂林/深圳)提供低成本智驾方案，小鹏(广州)是城市NOA先行者。比亚迪在深圳扎根20年，已形成从电池到整车的垂直整合。

31.6 投资者实地考察框架 — 四步筛选法

去产业集群考察不是走马观花，要有明确的分析框架。以下是经过验证的四步筛选法，帮助投资者快速判断一个产业集群的真实价值和风险。

第一步：供应链半径测试

到产业集群所在地，打开地图测量"30公里供应链圈"。问自己三个问题：

  1. 电池厂是否在30公里内？(电池占整车成本30-40%)
  2. 电机/电控配套是否就近？
  3. 模具/压铸/冲压件能否当日达？

实操方法：用高德地图搜索"电池""电机""模具"，看30公里半径内有多少家相关企业。长三角(常州/上海)可达100+家配套，中西部通常30-50家。配套密度每低一倍，物流成本增加5-8%。

第二步：政策可持续性评估

评估维度红旗信号(放弃)绿灯信号(考虑)
财政健康地方政府债务率>200%一般公共预算收入>200亿
政策连续性每年换一次招商政策NEV作为首位产业写进五年规划
土地供应工业用地已饱和预留5,000亩以上产业用地
人才储备周边无高校/无工程师本地有理工科院校+技校
退出通道已有企业大规模撤离上下游企业持续入驻
经验之谈：不要只听招商局说的政策，去工业园区旁边的餐馆吃顿饭，问老板"这附近工厂开得怎么样"。一线餐饮从业者往往比政府报告更早感知产业景气度。

第三步：实地调研清单

#调研对象关键问题信息来源

案例复盘：常州 vs 长沙的电池投资
2022年宁德时代选常州而非长沙建基地，表面看长沙土地/人工更便宜。但常州的优势在于：(1)距离上海/合肥的电池研发资源都在2小时车程内；(2)周围有理想/荣威等整车客户；(3)港口运输锂电材料成本低。最终结果：常州电池产值在三年内突破1,500亿。这个案例说明——产业集群的"网络效应"远比单项成本重要。
避坑指南：(1)警惕"零地价"陷阱——可能隐含对赌条件；(2)关注用电成本，NEV制造是耗电大户，电价差可达10%利润；(3)注意环保审批门槛，化工/电镀环节审批周期1-2年。

31.7 八大必去产业集群参观路线推荐

路线一：长三角NEV黄金三角(5天)

  1. D1 上海：特斯拉超级工厂(预约参观) + 蔚来NOPETech Day展厅
  2. D2 常州：宁德时代基地 + 理想汽车工厂 + 天合光能(光伏+储能)
  3. D3 合肥：蔚来中国总部 + 比亚迪合肥基地 + 联电芯片
  4. D4 宁波：吉利汽车研究院 + 拓普集团(底盘/内饰) + 旭升股份(压铸)
  5. D5 苏州：磊知智能(自动驾驶) + 汽车零部件产业园

考察重点：这条路线让你看到"整车+电池+芯片+零部件"的完整生态。注意观察每条产线的自动化程度和工人密度，这是判断制造业竞争力的核心指标。

路线二：珠三角智驾走廊(4天)

  1. D1 深圳：比亚迪总部(坪山基地) + 华为松山湖智驾实验室(预约) + 大疆车载
  2. D2 广州：小鹏汽车工厂 + 广汽埃安 + 希音供应链(跨境电商物流参考)
  3. D3 惠州：亿纬锂能工厂 + 电池材料配套园
  4. D4 东莞：德赛西威(智能座舱) + 智能制造示范基地

考察重点：珠三角的核心是"智能化基因"。注意感受互联网思维与传统制造的碰撞方式——这里的企业普遍在做"软件定义汽车"的实验。

路线三：中西部成本洼地(4天)

  1. D1 郑州：比亚迪郑州基地(全球最大的iPad代工转NEV) + 富士康
  2. D2 武汉：东风汽车转型基地 + 比亚迪武汉工厂
  3. D3 长沙：比亚迪长沙基地 + 湖南裕能(磷酸铁锂) + 山河智能
  4. D4 成都/重庆：长安汽车NEV平台 + 吉利成都基地 + 比亚迪成都

考察重点：中西部的核心竞争力是"成本+市场"双重优势。注意比较同样的产线在中西部的人工成本差异(通常比东部低25-35%)。

实战记录：某VC合伙人2024年产业考察日记片段
"在常州看宁德时代基地，3公里内找到5家配套供应商，物流成本几乎可以忽略。但到了长沙，最近的电池材料厂在100公里外，光是运输费就吃掉2%的毛利。产业不是孤立存在的，它需要一个'朋友圈'。"
"合肥最有意思——政府不是简单给钱，而是用'产业基金+场景开放'的模式。给蔚来输血的同时，开放了合肥的公交/出租NEV化场景，让蔚来自证商业化能力。这种'投资+对赌+场景'的组合拳值得每个地方政府学习。"
考察前准备清单：(1)下载企查查/天眼查，预设好要调研企业的股权穿透图；(2)打印各地"十四五"NEV产业规划文件(政府官网免费下载)；(3)准备企业拜访函，提前2周发当地工信局或园区管委会协调；(4)带卷尺/胎压计/充电宝——别笑，实地考察这些都有用；(5)关注"高工锂电""汽研网"等行业媒体，提前了解目标企业最新动态。
