电池材料深度 — 正极/负极/隔膜/电解液的"四方博弈"

锂电池由四大核心材料构成：正极(决定能量密度)/负极(决定储能能力)/隔膜(决定安全性)/电解液(决定离子传导)。这四者合计占电池成本的85%+。中国企业在四大材料领域均处于全球领先地位。

15.3 隔膜 — 安全的关键防线

类型代表企业厚度(μm)单价(元/m²)市占率
湿法基膜恩捷股份(全球35%+)/星源材质9-180.6-1.0~70%
干法基膜恩捷/星源材质/中材科技20-400.3-0.6~25%

隔膜是技术壁垒较高的环节。湿法隔膜更薄、孔隙率更高，适合高能量密度电池。涂覆(陶瓷/芳纶/PE)提升耐热性和安全性，是高端电池的标配。

15.6 正极材料选型 — LFP vs 三元实战指南

磷酸铁锂(LFP)选型要点

  1. 适用场景：A/B 级乘用车(续航 400-600km)、两轮车、储能。成本敏感型项目首选。
  2. 电芯选型参数：容量 2.8Ah-100Ah 方块电池或软包叠片。首次筛选看 3C 放电容量保持率(≥85% @ 3C)。
  3. 供应商对接流程：德方纳米 / 万润新能 / 龙蟠科技。先索取样品 → 做 3 次充放电循环 → 测 25°C 和 -10°C 容量衰减 → 确认批次一致性(CV ≤ 1.5%)。
  4. 常见坑：部分小厂为降低成本掺入廉价铁源，导致第一次循环效率只有 85% 以下， pack 后容量掉得很快。验收时要明确首圈效率 ≥ 90%。

三元材料(NCM/NCA)选型要点

  1. 适用场景：长续航车型(续航 700km+)、低温地区、高性能两轮车。对成本和重量都敏感的场合慎用。
  2. 选型路径：NCM523(性价比) → NCM622(均衡) → NCM811(高能量密度)。2024 年起主流新车型选 NCM622/721，少用 811（安全性顾虑）。
  3. 供应商对接：容百科技 / 当升科技 / 厦钨新能。重点考察 Ni 含量偏差（±0.5% 以内）、残留锂含量（≤1.0%）和粒径 D50（10-15 μm 最佳）。
  4. 高温烧结工艺窗口：三元材料烧结温度 850-1000°C，不同批次波动不超过 ±20°C。确认供应商有没有在线温控系统。

电解液配方选择 — 快充方案的加分项

配方类型典型配方适用场景代表供应商加价幅度
标准配方LiPF6 + EC/DMC/EMC (3:4:3) + VC 添加剂常规 NEV/两轮车天赐材料 / 新宙邦基准
快充配方上述 + FEC + LDTD + 高浓度 LiFSI神行超充/4C 以上快充新宙邦 / 瑞泰新材+15-25%

实操建议：电解液不是越贵越好。标准配方 + 优质隔膜就能满足 90% 的项目需求。只在明确有 4C+ 快充或 −20°C 低温场景时，才值得加钱上快充/低温配方。采购时认准天赐材料和新宙邦两家，占了全球 60%+ 份额，质量和交期最稳。

隔膜供应商选型 — 湿法 vs 干法怎么挑

  1. 湿法基膜：恩捷股份是绝对龙头(全球 35%+)。优势：薄(9-12μm)、孔隙率高(40-50%)、离子传导好。适合高能量密度电池。劣势：拉伸强度弱，需涂覆补强。
  2. 干法基膜：星源材质、中材科技。优势：成本低、力学性能好、热收缩率低。适合储能/两轮车/低端 LFP。劣势：厚度下限 20μm+，不利于高能量密度。
  3. 涂覆隔膜：陶瓷涂覆(SEO/Al2O3)是标配；芳纶涂覆适合高温场景(800V+ 电池)。采购确认：涂覆克重(单面 8-15g/m²)、附着力(划格法 0 级)。

电池四大材料 — 供应商对接 checklist

新项目材料选型 Checklist：

  1. 确定目标电池：容量(kWh)、电压平台(V)、放电倍率(C)、循环次数要求
  2. 根据目标选正极体系：LFP(成本优先) or NCM(能量密度优先)
  3. 选定负极：人造石墨(通用) / 硅碳(高能量密度) / 天然石墨(低成本)
  4. 隔膜厚度与涂覆方案匹配正极类型
  5. 电解液配方根据快充/低温需求选配
  6. 向 3+ 家供应商发 RFQ(询价单)，要求提供样品和 COA(质检报告)
  7. 小试 100-500 颗电芯 → 中试 2-5 kWh pack → 量产验证
  8. 签订年度框架协议，锁定折扣系数和调价机制
