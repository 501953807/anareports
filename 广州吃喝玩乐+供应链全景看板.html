2.1 美食地图

广式早茶

  • 核心品种: 虾饺 / 烧卖 / 叉烧包 / 肠粉 / 凤爪 / 蛋挞 / 流沙包 / 糯米鸡 / 红米肠 / 萝卜糕
  • 点都德 — 100+门店，全市密铺
  • 陶陶居 — 百年老字号，上下九旗舰店
  • 广州酒家 — 文昌南路总店，传统正宗
  • 莲香楼 — 太平南路，月饼/早茶双绝
  • 泮溪酒家 — 荔枝湾园林酒家，环境一流

人均 ¥50-120

烧腊

  • 核心品种: 深井烧鹅 / 豉油鸡 / 蜜汁叉烧 / 白切鸡 / 腊肠 / 烧肉
  • 炳胜公馆 — 黑叉烧名动全城，珠江新城店需排队
  • 万丰酒楼 — 烧鹅一绝，越秀老店
  • 惠福东路美食街 — 烧腊店铺最密集的一条街
  • 宝业路烧腊带 — 社区口碑店聚集

人均 ¥30-80

糖水 / 甜品

  • 核心品种: 姜撞奶 / 双皮奶 / 杨枝甘露 / 绿豆沙 / 芝麻糊 / 红豆沙 / 豆腐花

  • 玫瑰甜品 — 文明路，老西关手工糖水
  • 芳记小食 — 宝业路，炸牛奶/双皮奶

大排档 / 宵夜

  • 核心品种: 炒田螺 / 椒盐濑尿虾 / 干炒牛河 / 蒜蓉烤生蚝 / 煲仔饭 / 猪杂粥

  • 东晓南路 — 地道老广宵夜一条街

2.2 必游景点

历史文化

  • 陈家祠 — 岭南建筑巅峰之作，地铁1号线陈家祠站A出口
  • 南越王博物院 — 西汉南越王赵眜陵墓，解放北路
  • 光孝寺 — "未有南华先光孝"，六祖慧能剃度处
  • 六榕寺 — 苏轼题名"六榕"，花塔1000余年
  • 石室圣心大教堂 — 全花岗岩哥特式，东南亚唯一
  • 黄埔军校旧址 — 长洲岛，近代史发源地

现代地标

  • 广州塔（小蛮腰） — 600米，珠江南岸，可俯瞰全城
  • 珠江新城 — 花城广场 / 省博物馆 / 广州图书馆 / 大剧院
  • 海心沙 — 2010亚运会开幕式场馆，灯光秀最佳观景点
  • 周大福金融中心（东塔） — 530米，华南第一高楼
  • 西塔（广州国际金融中心） — 440米，珠江新城西翼

自然山水

  • 白云山 — "羊城第一秀"，摩星岭海拔382米，索道上下
  • 越秀公园 — 五羊雕塑 / 镇海楼 / 明代古城墙
  • 海珠湿地公园 — 城市中轴线上最大生态绿洲
  • 华南植物园 — 中科院下属，国家重点植物园
  • 荔湾湖公园 — 西关风情，荔枝湾涌重现

主题乐园

  • 长隆野生动物世界 — 全国最好，自驾小火车穿越
  • 长隆欢乐世界 — 垂直跌落过山车 / 十环过山车
  • 长隆水上乐园 — 吉尼斯认证全球最大水上乐园
  • 长隆国际大马戏 — 全球最大马戏表演，必看

地址: 番禺区迎宾路

交通: 地铁3号线 / 7号线汉溪长隆站
