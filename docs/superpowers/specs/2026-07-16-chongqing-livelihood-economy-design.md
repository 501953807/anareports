# 重庆产业全景看板·民生经济深度解剖设计方案

> **Goal:** 为"重庆产业全景分析与未来展望看板"新增民生经济深度解剖板块，从街头巷尾的行业微观视角透视重庆经济真实活力。
>
> **Tech Stack:** Pure HTML5 + CSS3 + ECharts 5.x + Font Awesome 6.5.1（与现有看板一致）

---

## 一、核心问题诊断

当前看板的问题不是数据不够多，而是**数据太宏观、缺少烟火气**。GDP 3.25万亿、汽车产量340万辆——这些数据重要，但看不到重庆的真实脉搏。真正决定一个城市经济活力的，是街头的火锅店、社区的小面馆、夜市的烧烤摊、农贸市场的菜贩、快递站的骑手。

**本次深化的核心原则：** 每个行业都要回答三个问题：
1. 这个行业的真实规模有多大？（精确数据）
2. 它养活了多少人？（就业/生计维度）
3. 它的未来会往哪里走？（趋势预判）

### 2.2 八大民生行业

| # | 行业 | 分析维度 | 发散思考（我们可能遗漏的点） |
|---|------|---------|--------------------------|
| 1 | **火锅经济** | 门店数、消费频次、区域密度、供应链产值、外地人vs本地人消费结构 | ①火锅底料工业化→调味品产业升级 ②花椒/辣椒种植带动山区脱贫 ③"火锅+文旅"复合消费模式 ④海底捞/小龙坎等外来品牌在渝生存困境 |
| 2 | **重庆小面** | 门店数、全产业链产值、带动就业、标准化瓶颈、中央厨房化 | ①小面是重庆最低门槛的创业入口（一碗面启动资金<5万） ②"小面师傅"→"火锅老板"的职业晋升路径 ③预包装小面（速食装）的工业化前景 |
| 3 | **文旅消费** | 游客消费结构、重点商圈客流、复游率、网红→长红转化率 | ①"8D魔幻城市"地理经济学对商业布局的独特影响（一层是街、十层还是街） ②雾都气候对室内消费(KTV/洗浴/火锅)的拉动 ③方言文化IP的内容经济价值 |
| 4 | **电商经济** | 直播基地、跨境电商、农村电商、平台企业在渝布局 | ①果园港+渝新欧的双通道跨境优势 ②"数字重庆"产业大脑对中小商户的赋能 ③农产品上行vs工业品下行的剪刀差 |
| 5 | **社区商业** | 农贸市场、便利店、早餐摊点、最后一公里配送、社区团购 | ①山城地形导致"最后一公里"配送成本比平原城市高30-50% ②"梯坎经济"——爬坡上坎形成的特色小店生态 ③老旧小区改造对社区商业的重塑 |
| 6 | **中医中药** | 道地药材(黄连/厚朴/金银花/独活)、种植基地、服务网络、大健康产业 | ①重庆是全國最大黄连产地(占70%+) ②中药材种植是武陵山区扶贫核心产业 ③"药食同源"与火锅/小面食材供应链的交叉 ④中医药+康养旅游的组合潜力 |
| 7 | **夜市夜经济** | 夜间消费占比、重点商圈夜客流、从业者规模、青年就业贡献 | ①重庆夏季高温(40°C+)催生"昼伏夜出"的消费文化 ②"宵夜文化"是重庆社交的核心场景 ③夜市经济对低技能劳动力的吸纳能力 |
| 8 | **个体商户生态** | 个体工商户总数、餐饮占比、存活率、融资困境、数字化转型 | ①"棒棒军"转型——第一代农民工的后半生 ②返乡创业潮：外出打工者带资金技术回乡开店 ③"夫妻店"模式vs连锁化的博弈 |

### 2.3 新增发散性洞察板块（我们原来没想到的）

除了上述8个行业，还需要增加以下**跨行业洞察**：

#### 2.3.1 山城地理经济学
- 山地地形对商业成本的特殊影响（配送成本+30-50%、多层商业体租金结构）
- "立体城市"带来的独特消费场景（洪崖洞的1楼是马路、11楼还是马路）
- 桥都经济：跨江桥梁对商圈分隔与连接的影响

#### 2.3.2 气候经济学
- 夏季40°C+高温→冷饮/凉虾/冰粉产业规模
- 冬季多雾少阳→火锅/洗浴/KTV等室内消费溢价
- 湿度大→中药材(除湿类)种植与消费的天然优势

#### 2.3.3 文化IP经济
- 重庆方言在短视频平台的传播力与变现能力
- "赛博朋克8D城市"的视觉IP对影视/摄影/打卡经济的影响
- 码头文化→江湖菜→市井火锅的文化传承链

#### 2.3.4 产业传导预测链
从民生行业推导未来带动性产业：
```
火锅/小面连锁化
  → 中央厨房/预制菜
    → 食品工业化
      → 冷链物流升级
        → 农业种植标准化(辣椒/花椒/生姜)
          → 调味品研发与品牌化
            → 食品安全检测服务业
```

## 五、数据准确性要求

1. **精确数据**：必须标注数据来源（统计年鉴/行业协会/政府公报），优先使用2024年及以后数据
2. **预测数据**：明确标注"预测值"或"估算"，说明假设条件
3. **无法核实的数据**：使用"约""估计"等措辞，不编造精确数字
4. **对比数据**：与成都/武汉/杭州等对标城市对比时，确保同口径
