# 广州吃喝玩乐+供应链全景看板 — 设计文档

**日期：** 2026-07-01
**类型：** HTML 全景看板（单文件）
**受众：** 旅游+商务兼顾（来广州的综合考察人员）
**风格：** 赛博朋克深色主题 + 霓虹强调色 + ECharts 数据可视化

---

## 1. 目标

构建一个单文件 HTML 全景看板，全面展示广州的吃喝玩乐推荐与全国供应链市场生态，服务于旅游+商务综合考察场景。内容详尽到街道级别标注供应链市场的功能场景，并辅以数据可视化呈现广州经济、跨境电商、批发市场的规模与趋势。

---

### 模块五：跨境电商与网商基地篇（含 ECharts 趋势图+平台对比图）

#### 5.1 广州跨境电商概况
- **全国地位**：跨境电商进出口额全国第一（2024年超3000亿元）
- **主要平台广州基地**：
  - Shein：海珠区/番禺区供应链基地（快时尚，日SKU超10万）
  - Temu：番禺/南沙仓配中心（全品类低价）
  - TikTok Shop：海珠区客村/大石直播基地（短视频+直播带货）
  - 速卖通（AliExpress）：天河区总部/仓储
  - Shopee/Lazada：白云区/番禺区跨境仓
  - Amazon：南沙自贸区跨境业务中心
- **政策优势**：南沙自贸区跨境电商综试区、1210/9610监管代码

#### 5.2 主要网商/电商基地
- **大石/客村片区（海珠区）**
  - 地址：大石大街/客村地铁站周边
  - 业态：直播电商基地、MCN机构、短视频拍摄棚

- **南浦岛（番禺区）**

  - 业态：跨境电商产业园、仓储物流中心
  - 特点：靠近南沙港，物流时效优势
- **琶洲片区（海珠区）**
  - 地址：琶洲大道/会展大道
  - 业态：电商平台总部（唯品会/希音/微信）、跨境电商孵化园

- **大石/南村（番禺区）**
