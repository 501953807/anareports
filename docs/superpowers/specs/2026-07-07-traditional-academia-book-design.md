# 中华古典学术理论全景书设计

## 目标

创建一套关于八卦、阴阳五行、中医理论、易经、道家、儒家六大主题的完整书籍和看板，采用实操+理论结合的风格，与现有7本书保持一致的赛博朋克深色主题。

## 架构

### 书籍结构（约32页）

**第一部分：八卦起源与演变**（4-5页）
- 八卦起源（河图洛书、伏羲画卦）
- 先天八卦与后天八卦
- 八卦象征体系（天地水火风雷山泽）
- 八卦在当代的应用（风水/命名/决策）

**第二部分：阴阳五行哲学**（5-6页）
- 阴阳理论的起源与发展
- 阴阳对立统一法则
- 五行（木火土金水）相生相克
- 阴阳五行在中医/风水/命理中的交叉应用

**第三部分：中医基础理论**（6-7页）
- 藏象学说（五脏六腑）
- 经络系统与穴位基础
- 气血津液理论
- 病因病机（六淫/七情/内生五邪）
- 中医诊断方法（望闻问切实操）
- 中医治疗手段（中药/针灸/推拿/食疗实操）

**第四部分：易经体系**（5-6页）
- 易经成书历史与版本演变
- 六十四卦详解（卦辞/爻辞）
- 占卜方法与解卦逻辑
- 易经哲学思想（变易/不易/简易）

**第五部分：道家理论**（4-5页）
- 老子道德经核心思想
- 庄子齐物论与自然观
- 道家修炼体系（内丹/外丹/导引）
- 道教组织发展与宫观文化

**第六部分：儒家理论**（4-5页）
- 孔子与论语核心思想
- 孟子性善论与仁政
- 宋明理学（程朱/陆王）
- 儒家伦理在当代的应用

### 看板（1页）

一张大图整合六大主题的理论框架、历史时间线、核心概念关系图。

## 实操特色

每页加入：
- **工具链清单**：经典书目推荐、在线课程、学习资源
- **操作步骤**：实践方法（太极练习步骤、穴位定位方法、占卜入门流程、冥想修炼步骤等）
- **准入条件**：学习路径建议（从入门到进阶）
- **收益路径**：理论学习的实际价值（养生/决策/文化修养）
- **自我保护**：辨别伪科学/迷信的方法
- **常见陷阱**：学习中容易误解的概念
