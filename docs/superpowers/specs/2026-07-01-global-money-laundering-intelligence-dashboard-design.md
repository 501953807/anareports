# 全球地下钱庄与洗钱全景风险预警看板 — 设计文档

**日期：** 2026-07-01
**类型：** HTML 全景看板（单文件）
**受众：** 金融合规/风险预警用途
**风格：** 赛博朋克深色主题 + 霓虹强调色 + ECharts 数据可视化

---

## 1. 目标

构建一个单文件 HTML 全景看板，全面展示全球地下钱庄及各类洗钱手法的运作机制、典型案例、执法数据和趋势预测，服务于金融风险合规场景。

---

## 2. 页面结构与模块

### 模块一：全局数据大字报（Hero Section）
- 全球洗钱规模：$800B-$2T/年（GDP 2-5%，FATF 估算）
- 链上犯罪总量：2023 年 $209亿（Chainalysis）
- 贸易洗钱 TBML：$6000亿-$1万亿/年
- 标签：数据来源标注（FATF / Chainalysis / FinCEN / Europol / UNODC）

### 模块三：洗钱渠道全景流程图
- **洗钱三阶段流程**：放置(Placement) → 分层(Layering) → 融合(Integration)
- **六大渠道卡片**（每个含：名称、步骤拆解、典型案例、涉及金额）：
  1. 地下钱庄对敲模式（国内 + 跨境）
  2. 哈瓦拉(Hawala)体系（中东/南亚）
  3. 虚拟币混币器（Tornado Cash/Blend/Wasabi）

  5. 贸易洗钱 TBML（虚报价格/幽灵货运）
  6. 离岸空壳公司（巴拿马文件模式/BVI/Cayman）

### 模块四：虚拟币洗钱专题

### 模块五：东南亚地下钱庄专题
- **流程图**：杀猪盘/诈骗 → 加密货币 P2P → OTC 跑分 → 地下钱庄对敲 → 境外账户

### 模块六：执法动态与合规建议
- **时间线**：2020-2026 重大执法行动（Tornado Sanction 2022/Blend Takedown 2025/US DOJ $12.6B 没收 2024）
- **趋势预测**：加密资产洗钱增长、AI 赋能反洗钱、跨境合作强化
- **合规建议**：针对金融机构的实操建议

## 4. 视觉规范

- **背景色**：#0a0e17（深空蓝黑）
- **卡片背景**：#111827（带半透明边框）
- **主文字**：#e2e8f0
- **次要文字**：#94a3b8
- **霓虹强调色**：
  - 红色（危险/犯罪）：#ef4444
  - 青色（数据/科技）：#06b6d4
  - 紫色（虚拟币）：#8b5cf6
  - 绿色（执法/合规）：#10b981
  - 橙色（警告）：#f59e0b
- **字体大小**：标题 28-36px，正文 14-16px
- **卡片圆角**：12px
- **阴影**：霓虹 glow 效果（box-shadow 带颜色扩散）
- **Hover 效果**：卡片上浮 + 边框发光
