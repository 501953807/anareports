# 重庆产业全景分析 — 设计文档

> **目标：** 全方位分析重庆当前及未来行业发展，形成一套 31 页 HTML 书籍 + 1 个独立看板
> **风格：** 赛博朋克深色主题，统一视觉体系，ECharts 数据可视化
> **受众：** 关注区域经济、产业投资、城市发展的读者

---

## 一、项目概述

### 1.1 背景
重庆作为中国西部唯一直辖市、西部大开发重要战略支点、"一带一路"与长江经济带联结点，其产业发展具有独特的样本价值。在经济下行周期、通缩压力下，重庆的优劣势分析需要基于真实数据，避免主观臆断。

### 1.2 范围
- **32 页书籍**（1 封面 + 31 内容页）：6 大部分 × 5 章 + 1 封面
- **1 个看板文件**：自包含 HTML，7 个 ECharts 实例
- **数据来源**：重庆统计年鉴 2025、国家统计局、重庆市 GDP 公报、成渝双城经济圈规划
- **时间跨度**：2015-2025 历史数据 + 2026-2030 预测

### 1.3 约束
- 纯静态 HTML，可部署到 GitHub Pages
- 赛博朋克深色主题，响应式设计
- 所有数据标注来源，不编造
- 宏观数据与区县民生并重

```

├── chongqing/                    # 新书目录
│   ├── index.html                # 封面 + TOC
│   ├── 01-demographic-overview.html
│   ├── 02-hukou-vs-resident.html
│   ├── ... (30 pages)
│   └── common/                   # （无，引用全局 common/）
├── common/                       # 公共资源（只读）
│   ├── css/book.css
│   ├── js/echarts.inline.js
│   └── js/navigation.js
├── 重庆产业全景分析与未来展望看板.html
└── index.html                    # 更新统计数据
```

**每页模板**：
- `<header class="chapter-header">` 包裹 part label + title + intro
- 5 段式结构：故事引入→概念解析→深入理解→实践应用→延伸阅读
- 表格、卡片、tip-box、case-box、步骤列表
- 至少 1 个 ECharts 图表
- footer prev/next 导航
- 现代侧边栏格式（sidebar-header + nav.toc）

**路径约定**：`../common/css/book.css`（subdirectory 模式）
