第六章：APP功能规划与开发团队配置

APP MVP版本(3-4个月)40-70万元，标准版(5-7个月)70-120万元，完整版(8-12个月)120-200万元。6-7人团队月成本10-14万。技术栈：Go/Java后端 + Swift/Kotlin双端 + Vue/React管理后台。

6.1 APP功能模块规划

模块功能优先级开发周期技术要点
设备配对BLE配对手环 + WiFi配药盒P0 必须2-3周BLE GATT协议 / WiFi SmartConfig
实时定位地图高德/百度地图API集成，实时轨迹回放P0 必须3-4周地图SDK + GCJ-02转换 + 轨迹插值
电子围栏多边形围栏设置、出入告警推送P0 必须2-3周地理围栏API + 多边形点包含算法
心率/血氧监测PPG数据可视化、异常阈值预警P0 必须2-3周时序数据图表 + 阈值规则引擎
跌倒检测告警IMU数据触发SOS推送P0 必须2-3周端侧AI模型 + 告警优先级路由
SOS紧急呼救一键拨号给预设联系人P0 必须1-2周Telephony API + 通讯录权限
用药提醒/药盒联动定时提醒、服药记录、家属通知P1 重要2-3周本地通知 + 云端调度 + BLE指令
家属多人共享多账号绑定同一老人设备P1 重要2-3周设备-用户多对多 + 权限管理
健康周报/月报数据聚合生成可视化报告P2 增值2-3周数据聚合 + PDF/图片生成
后台管理Web端运营人员管理后台P2 增值3-4周Vue/React + ECharts + RBAC

维度技术方案
后端Go(Gin) 或 Java(Spring Boot) + MySQL + Redis + MongoDB + RabbitMQ/Kafka
部署阿里云/腾讯云 ECS + Docker + K8s，CI/CD: GitHub Actions / GitLab CI
地图高德地图API(轨迹绘制/电子围栏) / 百度地图API(备选)
