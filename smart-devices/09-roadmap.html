第九章：8个月上市实施路线图 — 手环/药盒/APP并行推进

手环开发16-24周(4-6个月)，药盒16-22周(4-5.5个月)，APP MVP 3-4个月。三者并行推进，8个月实现双产品同步上市。

9.1 手环ODM开发全流程

阶段时间关键任务交付物
Phase 12周需求评审：功能清单/定位精度/续航目标/IP67防水/SIM卡方案PRD文档
Phase 22-3周ODM方案选型：润科/杰理现有方案，评估MCU+定位+传感器+屏幕方案选型报告
Phase 33-4周ID/MD设计：适老化设计(大字体/大图标/实体SOS/硅胶表带/防拆卸表扣)3D渲染图+结构图
Phase 43-4周开模+手板验证：3D打印→防水测试→跌落测试(1米)功能手板x10+
Phase 52-3周EVT试产：工程验证测试(功能/射频/电池续航)EVT报告
Phase 62-3周DVT设计验证：高低温(-10~50°C)/盐雾/老化测试DVT报告
Phase 72周PVT小批量 → MP量产量产出货

9.2 药盒ODM开发全流程

阶段时间关键任务交付物
Phase 12周需求评审：药盒容量/分药机构方案/WiFi+BLE/续航目标PRD文档
Phase 22-3周ODM方案选型：ESP32-C3 + 步进电机分药 + 红外/光电检测方案选型报告
Phase 33-4周结构设计：食品级塑料药仓/防儿童锁/语音喇叭/LED指示3D结构图
Phase 43-4周开模+手板：3D打印验证分药机构可靠性(目标>10,000次循环)功能手板x10+
Phase 54-6周EVT/DVT/PVT：可靠性验证+小批量量产出货
