干蒸烧卖1 笼¥32/笼猪肉+虾仁双拼最经典
流沙包1 笼¥18/笼咸蛋黄奶酥馅，趁热掰开
艇仔粥1 碗¥22/碗配花生+鱼片+瘦肉+蛋丝
实战要点
  • 茶位费：广州各店 ¥2-10/位不等，茶壶到即斟，不必客气
  • 推车阿姨：传统茶楼有小推车现场推，按笼计价；点心单叫制通常更便宜但品种少
  • 排队时段：周末 8:30-9:00 到号最快，10:30 后基本不用等
  • 扫码点单：新式茶楼普遍支持微信小程序点单，手机下单比等服务员快一倍

三、批发市场砍价三步法

  1. 货比三家（10-15 分钟）：先到周边同类档口逛一圈，记下 2-3 家同品质报价底线。拿手机拍照记价，方便回来时直接引用。
  2. 压价开场（2 分钟）：用"我拿 X 件样品，做不好下次补单"当话术。首价往低报 15-20%，给双方留谈判空间。
  3. 成交锁定（5 分钟）：谈妥后立刻下定金（通常 20-30%），要求写清规格、交期、退换条件。档口收据要有盖章或负责人签字。
防坑提醒
  • 以次充好：样品和批量供货颜色/材质有差异？先拿大货样确认再付尾款
  • 混装套路：服装/皮具混入低质库存货？开箱抽检比例不低于 10%
  • 物流转嫁：批发市场报价通常不含物流，问价时务必问"包不包送货"

四、跨境电商卖家入门流程

广州是全国跨境电商第一大城。以下是在广州从零起步的典型步骤：

Step 1-3：选品 + 开店

  • 从 十三行（快时尚女装）、沙河（低价爆款）、白马（中高端女装）拿样
  • 注册 Temu / SHEIN / TikTok Shop 卖家账号，需要营业执照
  • 在 天河科韵路 一带找代办公司，半天可下照（约 ¥500）

Step 4-6：发货 + 售后

  • 用 白云区太和 跨境仓，一件代发 ¥3-5/票
  • 头程走南沙港/白云机场空运，海运走黄埔港
  • 退货处理可选 花都电商产业园，质检重新上架损耗低 30%
