7.2 终极推荐 — 银发经济为首选

维度理由
人口确定性中国65岁以上人口从1.75亿(2020)→3.7亿(2035)，15年翻一倍。这是不可逆的趋势。
政策确定性十四五规划明确将银发经济列为战略性产业，长期护理保险试点扩大，医保覆盖加速。
支付能力60后/70后新一代老年人拥有房产+储蓄+养老金，购买力远超日本1990年的老年群体。
供给缺口持证护理人员缺口超1000万，老年医药/器械/服务严重不足，市场集中度极低。
抗周期性医疗健康是刚性需求，不受经济周期影响。即使悲观情景下也保持12%+增长。
日本验证日本1990-2020年，医药/护理/老年用品是少数持续增长的行业，跑赢GDP增速5-8倍。

实操指南：个人财务规划的详细执行方案

核心思路
前面的分析告诉你"做什么"，本部分告诉你"怎么做"。以下提供从开账户到日常管理的完整操作指南。

第一步：股票账户开立与ETF选择

  1. A股账户：选择佣金低的券商（目前主流互联网券商佣金约万分之2.5）。推荐使用支持网格交易/条件单的APP，方便自动执行定投策略。
  2. 港股通：A股账户满足50万资产门槛后可开通港股通。或者通过场内港股ETF间接投资港股（如恒生ETF（513660）、恒生科技ETF（513180））。
  3. 定投ETF选择顺序：第一梯队：沪深300ETF+中证500ETF（宽基打底）；第二梯队：红利ETF+港股高息ETF（收息为主）；第三梯队：纳指ETF(QDII)+日经ETF(QDII)（分散国别风险）。
  4. 黄金ETF：直接购买黄金ETF（518880）或场外黄金ETF联接基金，避免实物黄金的保管烦恼。

第二步：房地产投资策略——城市分级实操指南

日本房地产投资案例
东京都心部（港区/千代田区/涩谷区）的房地产泡沫后跌幅仅30-40%，远低于全国平均的60-70%。而且从2013年安倍经济学开始后，东京都心部房价在10年内恢复到泡沫时期水平并创历史新高。关键逻辑：在资产泡沫破裂后，人口和资本加速向核心城市集中。在中国，这个规律同样适用——北上广深的核心区房产在调整后依然保值为最佳选择。

第三步：个人税务优化实操

  1. 专项附加扣除：确保在个人所得税APP中填写全部6项（子女教育、继续教育、大病医疗、住房贷款利息/租金、赡养老人、3岁以下婴幼儿照护）。预计每年可节税2000-24000元不等。
  2. 个人养老金：每年最多存入12000元，可在当年应纳税所得额中扣除。如果你的个税税率在10%及以上，每年可省税1200-5400元。
  3. 年终奖单独计税：2027年底前，年终奖可以选择单独计税或并入综合所得，建议两个方案都算一下，哪个省税选哪个。
  4. 基金持有期限：持有ETF/公募基金超过1年免税；持有股票超过1年分红免税。合理安排交易频率可以节省税费。

附：个人投资决策检查清单（打印贴在电脑旁）

  1. [ ] 我是否清楚这笔投资的底层资产是什么？（不是"听说能赚"就买）
  2. [ ] 我是否了解这个产品的全部费用？（管理费/托管费/申购费/赎回费）
  3. [ ] 如果明天这个产品跌了30%，我会怎么办？（提前想好止损/持仓策略）
  4. [ ] 我是否有足够应急资金后再投资？（至少6个月生活费的现金储备）
  5. [ ] 我是否咨询了独立第三方意见？（不要只听券商/银行客户经理的推荐）
  6. [ ] 我的资产配置是否过度集中？（单一行业/单一国家不超过30%）
  7. [ ] 我是否记录了这笔投资的理由？（写下决策日志，便于日后复盘）
  8. [ ] 我的收入来源是否多元化？（不依赖单一工资收入，考虑副业/被动收入）
