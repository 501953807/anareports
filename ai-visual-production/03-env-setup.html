二、免费工具配置指南

2.1 通义万相 — 零门槛图像生成

  1. 注册阿里账号
    访问 aliyun.com，使用手机号注册阿里云账号。首次注册赠送一定额度的免费算力资源。
  2. 打开通义万相
    在阿里云控制台搜索"通义万相"或直接访问 wanx.aliyuncs.com。也可以直接使用"通义"APP（移动端），功能完全相同。
  3. 选择图像生成模式
    通义万相提供多种模式：文生图、图生图、风格迁移、人物写真。新手建议从"文生图"开始。
  4. 输入提示词并生成
    使用中文提示词即可（通义万相对中文理解最好）。参考第2章的提示词公式，输入包含主体、风格、光照的描述。点击生成后等待10-30秒即可获得结果。
  5. 下载与保存
    生成满意后可以右键保存图片。免费版每日有一定生成额度限制，通常足够日常练习使用。

2.2 可灵Kling — 免费视频生成

  1. 注册快手账号
    访问 klingai.com，使用手机号注册。新用户赠送大量免费积分。
  2. 进入可灵AI创作平台
    可选择"图像生成"或"视频生成"模式。视频生成功能目前全球领先，质量评分9.2/10。
  3. 图像生成视频
    推荐工作流：先在通义万相/Midjourney中生成高质量静态图，然后在可灵中选择"图生视频"模式。这种方式比直接文生视频效果更好、可控性更强。
  4. 视频参数设置
    可选参数包括：时长（5秒或10秒）、分辨率（720p或1080p）、运动强度（低/中/高）。新手建议先用默认参数体验。
  5. 每日免费额度
    可灵每日赠送约60积分，生成一段5秒1080p视频消耗约10积分。也就是说每天可以免费生成约6段短视频。

2.3 Stable Diffusion + ComfyUI — 本地部署指南

  1. 安装Python环境
    下载并安装Python 3.10.x（推荐版本，兼容性最好）。安装时务必勾选"Add Python to PATH"。
    ⚡ 提示：Windows用户建议从 python.org 下载安装包；Mac用户使用Homebrew：brew install python@3.10
  2. 克隆ComfyUI仓库
    打开终端/命令行，执行以下命令： git clone https://github.com/comfyanonymous/ComfyUI.git
  3. 安装依赖
    cd ComfyUI
    pip install -r requirements.txt
  4. 下载基础模型
    从 huggingface.co 或 civitai.com 下载SD 1.5或SDXL基础模型，放入 ComfyUI/models/checkpoints/ 目录。
  5. 启动ComfyUI
    python main.py
    启动后浏览器自动打开 http://localhost:8188。
  6. 安装自定义节点
    推荐使用ComfyUI Manager（一键安装管理器）。克隆到 ComfyUI/custom_nodes/ComfyUI-Manager，重启ComfyUI后即可在菜单中搜索安装各种自定义节点。
Mac用户特别提示

M系列芯片（M1/M2/M3）Mac可以直接运行ComfyUI，利用Apple Silicon的统一内存架构。虽然速度不如NVIDIA GPU，但8GB以上内存的Mac可以运行SDXL模型。启动命令相同：

python main.py --force-fp16

五、多学派交叉解读

公共物品理论视角
免费的AI工具正在成为数字时代的"公共基础设施"——类似于道路、电力和互联网。传统上，视觉创作需要昂贵的专业设备（相机、灯光、剪辑工作站），这些构成了"准公共物品"的门槛。现在，通义万相和可灵这样的免费工具将创作基础设施"公共化"，使任何人都能使用世界级的AI能力。这类似于电力普及如何 democratize 了制造业——AI正在 democratize 创意生产。
马克思主义政治经济学视角
免费工具的"免费"是有代价的——用户在用数据换取服务。当你使用通义万相时，你的提示词和生成结果都成为了训练数据的来源。这是一种"数字地租"关系：平台拥有基础设施（服务器、模型），创作者使用基础设施并贡献数据，平台从中获取价值。然而，这也产生了新的矛盾——当免费工具足够好时，创作者可能减少对付费工具的依赖，迫使平台寻找新的盈利模式。
演化经济学视角
免费工具的存在加速了整个生态系统的演化速度。当一个新工具出现并免费提供时，它会迅速吸引大量用户，这些用户产生的使用数据和反馈又加速了模型的迭代改进。这是一个正反馈循环：免费 → 用户增长 → 数据积累 → 模型改进 → 更多用户。这种"演化加速器"效应使得AI影像工具的发展速度远超以往任何创意技术。

六、发散引申链

免费工具普及 → 创作民主化 → 内容供给爆炸 → 注意力经济加剧 → 新筛选机制 → 个人品牌溢价
