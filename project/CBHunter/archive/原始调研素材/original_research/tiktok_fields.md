**完全被阻止**。导航到 `/promotion` 或相关营销URL时，页面返回：

```
无权限
您暂无此页面权限，不可进行任何操作。请联系主账号管理员获取相应权限。谢谢！
```

## 七、平台特色功能

### 7.1 SPS 卖家体验分 (Seller Performance Score)

- TikTok Shop 的核心卖家评分体系
- 影响 CRM 功能解锁（需 SPS ≥ 3.5）
- 可能还影响商品排名、活动报名资格等
- 类似 Shopee 的店铺评分但命名和计算方式不同

### 7.2 聊天营销 (Chat Marketing / CRM)

- 基于买家对话的主动触达系统
- 支持一次性计划和自动计划两种模式
- 模板化运营（分享商品、直播提醒、购物车挽回等）
- 与 TikTok 的社交/直播生态深度整合

### 7.3 联盟营销 (Affiliate Program)

- 独立模块入口：`/affiliate/landing`
- 跨境账号当前无权限访问
- 类似 Shopee 的联盟营销佣金体系

### 7.4 直播和视频生态

- 侧边栏有独立的"直播和视频"模块
- CRM中有直播相关的自动触达模板
- 与 TikTok 内容生态（短视频、直播）深度绑定
- 这是 TikTok Shop 相对于 Shopee 的核心差异化能力

### 7.5 权限体系

- 主账号/子账号权限管理
- 跨境店铺 vs 本地店铺权限差异
- 部分高级功能有门槛要求（SPS评分、GMV等）
- 无权限时显示统一的中文提示（即使界面可能是英文）

### 8.3 技术实现建议

**SPA渲染适配**:
- TikTok Shop 后端为SPA架构，chrome-devtools snapshot 无法可靠捕获动态内容
- CBHunter 可能需要通过 DOM 扫描或 API 逆向来获取字段数据
- 建议使用 JavaScript evaluation 辅助采集

**权限感知**:
- 检测"无权限"页面并给出友好提示
- 区分跨境/本地店铺的不同权限集
- 记录SPS门槛等解锁条件

**只读优先**:
- 大部分功能只需读取数据
- CRM聊天计划、营销活动涉及真实消息发送，必须避免自动操作
- 订单发货等操作需手动确认

**国际化支持**:
- 多语言界面（中文后台 + 各站点本地语言）
- 多货币显示（PHP/SGD/MYR/THB/VND/IDR/GBP/USD）
- 时区自动转换

## 九、研究总结

### 9.1 关键发现

1. **TikTok Shop Seller Center 是SPA架构**
   - 商品列表、商品创建等页面无法通过常规snapshot捕获
   - 需要DOM扫描或API逆向方法补充采集

2. **跨境店铺权限受限明显**
   - 营销中心（promotion）、财务模块、联盟营销均返回"无权限"
   - 可能与Shopee不同，Shopee跨境卖家可使用大部分营销工具

3. **SPS评分体系是核心门槛**
   - 影响CRM功能解锁（SPS ≥ 3.5 + GMV > 0）
   - 可能还影响商品排名和活动报名

4. **聊天营销是TikTok差异化功能**
   - 一次性计划 + 自动计划的模板化体系
   - 与直播、短视频生态深度绑定
   - 购物车挽回、结账挽回等自动化触达

5. **多站点统一管理**
   - 顶部栏直接切换站点
   - 跨境统一入口（tiktokshopglobalselling.com）
   - 各站点货币、类目、支付方式独立
